## Advanced Calculator

A Python desktop application built with Tkinter for basic and scientific calculations.

---

### Key Features

* **Basic Math**: Performs addition, subtraction, multiplication, and division.
* **Scientific Tools**: Includes square roots, exponents, factorials, and logs.
* **Trigonometry**: Sine, Cosine, and Tangent functions with Degree/Radian switching.
* **Quadratic Solver**: Calculates roots for $ax^2 + bx + c = 0$ using the `Eq` button.
* **Binary Converter**: Switches numbers between Decimal and Binary formats.
* **Shift Key**: Accesses secondary functions like Pi ($\pi$) and comma inputs.

---

### How to Run

1.  **Install Python**: Ensure Python 3.x is installed on your system.
2.  **Run Script**: Execute the file from your terminal:
    ```bash
    python "Advanced Calculator.py"
    ```

---

### Using the Equation Solver

1.  Enter three numbers separated by commas (e.g., `1, -5, 6`).
2.  Use **Shift** + **)** to type the commas.
3.  Press the **Eq** button to see the result.
